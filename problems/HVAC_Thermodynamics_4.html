HVAC & Refrigeration · Thermodynamics · Problem 4
Problem Statement
What is the density of propane at 77°F and atmospheric pressure?
Answer Choices
A.0.003 lb/ft³
B.0.02 lb/ft³
C.0.1 lb/ft³ ✓
D.60 lb/ft³
Solution
Answer: C
1. Identify given information: Propane (C₃H₈) at 77°F and atmospheric pressure (14.696 psia). Find density. 2. Determine the molecular weight of propane: M = 3(12) + 8(1) = 44 lbm/lb·mole. 3. Apply the ideal gas law: PV = nRT, or rearranged for density: ρ = PM / RT. 4. Convert temperature to absolute scale: T = 77°F + 459.67 = 536.67 °R. 5. Calculate density: ρ = P × M / (R_universal × T) = (14.696 × 144) × 44 / (1545 × 536.67) ≈ 0.1 lb/ft³. (Note: P must be in lbf/ft² if using R = 1545 ft·lbf/(lb·mole·°R).) 6. Select the closest answer: ≈ 0.1 lb/ft³ → Answer C.
This problem tests the application of the ideal gas law to find the density of a gas at specified conditions. The ideal gas law PV = nRT can be rearranged to solve for density. Starting from PV = nRT, we can write n = m/M (mass divided by molecular weight), so PV = (m/M)RT. Rearranging: P = (m/V)(RT/M) = ρRT/M. Solving for density: ρ = PM / (RT). For propane (C₃H₈): - Molecular weight M = 3 × 12.01 + 8 × 1.008 ≈ 44 lbm/lb·mole - Pressure P = 14.696 psia = 14.696 × 144 = 2,116.2 lbf/ft² - Temperature T = 77 + 459.67 = 536.67 °R - Universal gas constant R = 1,545 ft·lbf/(lb·mole·°R) Substituting: ρ = (2,116.2 × 44) / (1,545 × 536.67) = 93,113 / 829,155 ≈ 0.112 lb/ft³ This is closest to 0.1 lb/ft³ (Answer C). A sanity check: propane is heavier than air (M = 44 vs 29 for air), and air density at standard conditions is about 0.075 lb/ft³. So propane being slightly denser at ~0.1 lb/ft³ makes physical sense. The answer of 60 lb/ft³ would be a liquid density — way too high for a gas. And 0.003 lb/ft³ would be unreasonably light, even lighter than hydrogen.

⭐ Key Takeaways:
• The ideal gas law is a workhorse equation — know it in multiple forms, especially ρ = PM/(RT) for density calculations.

• Unit consistency is critical: if R is in ft·lbf/(lb·mole·°R), then pressure must be in lbf/ft², temperature in °R, and molecular weight in lbm/lb·mole.

• Always do a sanity check. Gas densities at atmospheric pressure are on the order of 0.01–0.1 lb/ft³. If you get 60 lb/ft³, that's a liquid — something went wrong.

📊 Using Formulas vs Default Values & Property Tables
OH 115 · Mar 2, 2026
Q: We used Δh = c_p ΔT to find enthalpy of water. What stops us from using that same formula to find Δh between superheated steam and subcooled water? Is it because they're two different states?
Dan: It's more than just being different states. Think about the T-h diagram: when you heat ice, the temperature rises (sensible heating). Then at the melting point, temperature stays constant while phase changes (latent heat). Then liquid water heats up (sensible again). Then at boiling, temperature holds constant again (more latent heat). Then steam heats up (sensible once more). The formula Q = mc_pΔT only works for sensible heating — within a single phase along one of those sloped lines. If you try to span across a phase change boundary, you miss all the latent heat involved in the transition. Plus, the c_p values are different for ice, liquid water, and steam. To cross phase boundaries, you need to use enthalpy values (mΔh) directly from the steam tables, which capture everything — sensible and latent.

Q: The reference handbook mentions a typical default value for the heat transfer coefficient h = 1.2. This is close to the 1.3 we calculated using the full equation. When is it reasonable to use the typical value vs doing the full calculation?
Dan: If the problem gives you enough data to calculate the actual value, you should probably do that — it's more precise and you can be confident it applies to your specific situation. Use the default/typical value only when there isn't enough information to calculate it directly. You could save time with the default, but you'd sacrifice confidence that it's correct for that case. For things like kinematic viscosity at different temperatures: check how much the property changes over your temperature range. If it's 5-10%, you're probably fine either way. If significantly different, then use the table value at the exact temperature.

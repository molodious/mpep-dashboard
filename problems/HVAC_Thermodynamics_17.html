Problem 17 — Reversed Carnot Refrigeration (Work Input)
Thermodynamics · Reversed Carnot Cycle · Refrigeration COP
The cold and hot reservoirs of a reversed Carnot refrigeration cycle are −20°F and 80°F, respectively. 3,000 Btu/hr are absorbed from the cold reservoir. What is the work input?
A. 160 W
B. 200 W  ✓
C. 700 W
D. 1100 W
Answer: B — 200 W
Quick Overview
  1. Identify the reversed Carnot cycle (refrigeration) and given temperatures: Tcold = −20°F, Thot = 80°F, Qcold = 3,000 Btu/hr.
  2. Convert temperatures from °F to absolute (Rankine): add 459.67 to each °F value.
  3. Calculate the Carnot COP for refrigeration: COP = T_cold / (T_hot − T_cold)
  4. Use the COP definition to solve for work input W: W = Q_cold / COP
  5. Convert W from Btu/hr to Watts using the conversion factor 1 Btu/hr = 0.29307 W.
  6. Compare to answer choices and select the closest value.
Key Takeaways
  • Always use absolute temperatures (Rankine or Kelvin) in Carnot efficiency calculations — using °F or °C directly will give wrong answers.
  • Refrigeration COP can be greater than 1, which confuses many students; it means you move more heat than the work you put in — the refrigeration cycle leverages the temperature difference to transfer heat, not generate it.
  • Unit conversion between Btu/hr and Watts shows up repeatedly on the PE exam — memorize the conversion factor: 1 Btu/hr = 0.29307 W (or equivalently, 1 W = 3.412 Btu/hr).
  • The reversed Carnot cycle sets the theoretical upper limit for any real refrigeration system — no real device can exceed Carnot COP at the same operating temperatures.

Full Solution
Answer: B — 200 W

Step 1: Identify the given information

This is a reversed Carnot refrigeration cycle — the most efficient possible refrigeration cycle operating between two temperature reservoirs. We are given:

  • Cold reservoir temperature: Tcold = −20°F
  • Hot reservoir temperature: Thot = 80°F
  • Heat absorbed from cold reservoir: Qcold = 3,000 Btu/hr
  • Find: Work input W (in Watts)

Step 2: Convert temperatures to Rankine

Carnot COP formulas require absolute temperatures. In the English system, convert °F to Rankine by adding 459.67:

T_cold = −20°F + 459.67 = 439.67 °R T_hot = 80°F + 459.67 = 539.67 °R

Note that Thot − Tcold = 539.67 − 439.67 = 100 °R. The even 100°R difference is a convenient feature of this problem.

Step 3: Calculate the Carnot COP for refrigeration

For a reversed Carnot (refrigeration) cycle, the COP is:

COP_Carnot = T_cold / (T_hot − T_cold) COP_Carnot = 439.67 / (539.67 − 439.67) COP_Carnot = 439.67 / 100 = 4.397

A COP of 4.397 means the refrigerator moves 4.397 Btu of heat for every 1 Btu of work input. This is greater than 1, which is normal and expected for refrigeration cycles.

Step 4: Solve for work input W

The COP definition relates Qcold, W, and COP:

COP = Q_cold / W ⇒ W = Q_cold / COP W = 3,000 Btu/hr / 4.397 W = 682.3 Btu/hr

Step 5: Convert Btu/hr to Watts

The answer choices are in Watts, so convert using the standard conversion factor:

1 Btu/hr = 0.29307 W W = 682.3 Btu/hr × 0.29307 W/(Btu/hr) W ≈ 200 W

Step 6: Select the answer

200 W matches answer choice B exactly. The other choices (160 W, 700 W, 1100 W) would result from temperature conversion errors (not using Rankine) or COP formula errors (using Thot instead of Tcold in the numerator).

✓ Final Answer: B — 200 W

🎓 Key Insights for This Problem
  • The reversed Carnot cycle is the theoretical maximum efficiency for refrigeration — any real system operating between the same two temperatures will have a lower COP.
  • Always convert to absolute temperature (Rankine) before applying COP formulas — substituting °F values directly into the Carnot expression gives a physically meaningless result.
  • The COP for refrigeration uses the cold reservoir temperature in the numerator: COPref = Tcold / (Thot − Tcold). Using Thot in the numerator gives the heat pump COP instead.
  • Unit conversion between Btu/hr and Watts is a common exam trap — the problem is set up so that the work in Btu/hr (682 Btu/hr) does not match any answer choice; you must convert to Watts to find 200 W.
  • The Carnot COP gives the maximum possible COP for any device operating between the given temperatures — real systems are always lower due to irreversibilities.
  • Know the difference: COPrefrigeration = Tcold / (Thot − Tcold) vs COPheat pump = Thot / (Thot − Tcold) — and that COPHP = COPref + 1.

Problem 17

The cold and hot reservoirs of a reversed Carnot refrigeration cycle are −20°F and 80°F, respectively. 3000 Btu/hr are absorbed from the cold reservoir. What is the work input?

A 160 W
B 200 W ✓
C 700 W
D 1100 W
Quick Overview

This problem tests your ability to apply the Carnot coefficient of performance (COP) for a refrigeration cycle and convert between Btu/hr and Watts. The key concept is computing COP using absolute temperatures (Rankine), then using it to find the required work input from the given cooling load.

Full Solution
Step 1

Convert to Absolute Temperature (Rankine)

The Carnot COP formula requires absolute temperatures. Convert °F to °R by adding 459.67:

T_cold = −20°F + 459.67 = 439.67 °R T_hot = 80°F + 459.67 = 539.67 °R
Step 2

Carnot COP for Refrigeration

The COP of a Carnot refrigerator is defined as:

COP_ref = T_cold / (T_hot − T_cold) COP_ref = 439.67 / (539.67 − 439.67) COP_ref = 439.67 / 100 COP_ref = 4.397
Step 3

Calculate Work Input

By definition, COP = Q_cold / W, so:

W = Q_cold / COP W = 3000 Btu/hr ÷ 4.397 W = 682.3 Btu/hr
Step 4

Convert to Watts

Use the conversion factor 1 W = 3.412 Btu/hr:

W = 682.3 Btu/hr ÷ 3.412 Btu/hr per W W ≈ 200 W
✅ Answer: B) 200 W
Concept Deep-Dive

This problem draws on several foundational thermodynamic principles that appear frequently on the PE exam:

Reversed Carnot Cycle

A Carnot cycle operating in reverse acts as an ideal refrigerator or heat pump. It sets the theoretical maximum COP for any cycle operating between two temperature reservoirs.

COP: Refrigeration vs. Heat Pump

COP_ref = T_cold / (T_hot − T_cold). COP_HP = T_hot / (T_hot − T_cold). Note: COP_HP = COP_ref + 1. Know which one the problem asks for!

Absolute Temperature Scales

Carnot COP requires absolute temps. For US customary: °R = °F + 459.67. For SI: K = °C + 273.15. Never use °F or °C in COP formulas.

Unit Conversions

1 W = 3.412 Btu/hr. This conversion is essential when the problem mixes IP cooling loads with SI answer choices — a common PE exam pattern.

Common Pitfalls

⚠ Using °F instead of °R: Plugging −20 and 80 directly into the COP formula gives a nonsensical negative COP. Always convert to absolute temperature first.
⚠ Confusing COP_ref with COP_HP: If you use T_hot in the numerator (heat pump COP = 5.397), you'll get a smaller work input (~163 W) — close to choice A. Read the problem carefully.
⚠ Forgetting the unit conversion: If you stop at 682 Btu/hr and pick the closest answer, you might select C or D. The answer choices are in Watts — always check units.
Key Takeaways
  • Always convert to absolute temperature (Rankine or Kelvin) before using Carnot COP equations.
  • COP_refrigeration = T_cold / (T_hot − T_cold) — the cold reservoir temperature is in the numerator.
  • COP_heat pump = COP_refrigeration + 1 — know the difference and read what the problem asks for.
  • Memorize 1 W = 3.412 Btu/hr (or 1 ton = 12,000 Btu/hr = 3,517 W) for quick conversions on exam day.
  • When answer choices use different units than the problem, the conversion step is part of the test — don't skip it.

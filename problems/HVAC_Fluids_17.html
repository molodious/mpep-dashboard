Problem

Which of the following is not a reason for pumps to be placed in parallel?
Answer Choices
A ✓ To increase pressure
B To increase volume flow rate
C To improve efficiency
D To add redundancy
Solution
✓ Correct Answer: A — To increase pressure
⚡ Quick Overview
  1. Read carefully — the question asks for what is NOT a valid reason for placing pumps in parallel. You need the one wrong answer, not the right ones.
  2. Recall the fundamental behavior of parallel pumps: they share the same suction and discharge headers, so each pump adds its flow to the total at the same system head.
  3. Choice B — increasing volume flow rate — IS the primary reason to use parallel pumps. The combined flow curve is obtained by adding flows at each head value. Valid reason → eliminate B.
  4. Choice D — adding redundancy — IS a major practical reason. If one pump trips offline, the remaining pump(s) keep the system running. Valid reason → eliminate D.
  5. Choice C — improving efficiency — IS a valid reason. Running two smaller pumps near their Best Efficiency Point (BEP) can be more efficient than one oversized pump running far off its BEP. Valid reason → eliminate C.
  6. Choice A — increasing pressure/head — is what series pumps do, not parallel. Parallel pumps do NOT add head; they add flow. This is NOT a valid reason → Answer A.
📋 Full Solution

Step 1 — Understand the two fundamental pump configurations.

When multiple pumps are combined in a system, there are two standard configurations: parallel and series. The key distinction lies in what each configuration adds to the system — flow or head.

Configuration How Connected What It Adds Pump Curve Construction
Parallel Same suction & discharge headers; pumps run side-by-side Flow (Q) at the same head Add flow rates horizontally at each head value
Series Discharge of one feeds inlet of the next; pumps run in-line Head (H) at the same flow Add heads vertically at each flow rate value

This distinction is the entire key to this problem. The question asks which choice is not a reason for using parallel pumps — and the answer is the one that describes what series pumps do instead.

Step 2 — Evaluate each answer choice.

Choice A: To increase pressure
Increasing system pressure or head is accomplished by connecting pumps in series. In a series arrangement, the discharge of the first pump feeds directly into the suction of the second pump. Each pump adds its head to the total, so the combined head at any given flow equals the sum of the individual pump heads. Parallel pumps, by contrast, operate at the same head point — they do not stack head. Therefore, "to increase pressure" is NOT a valid reason for parallel pumping. This is the correct answer.

Choice B: To increase volume flow rate
This IS the primary and most fundamental reason to use parallel pumps. When two identical pumps operate in parallel, the combined pump curve is constructed by adding their individual flow rates at each head value. The result: the system can deliver twice the flow (for two identical pumps) at any given head. Parallel pumping is the go-to solution when the required flow rate exceeds what a single pump can provide. Valid reason — not the answer.

Choice C: To improve efficiency
This IS a valid — though secondary — reason. Every centrifugal pump has a Best Efficiency Point (BEP), the flow and head combination at which it operates most efficiently. A single large pump sized for peak demand will often run well off its BEP during partial-load conditions, wasting energy. By instead using two smaller pumps in parallel, the system can run one pump alone during low-demand periods (keeping that pump near its BEP) and activate the second pump when demand increases. This staging strategy can yield meaningful efficiency gains in real HVAC systems. Valid reason — not the answer.

Choice D: To add redundancy
This IS a valid and very common practical reason — especially in mission-critical HVAC applications like data centers, hospitals, and large commercial buildings. If one pump fails or requires maintenance, the other pump(s) in the parallel bank continue to supply flow, preventing a complete system shutdown. Redundancy (also called N+1 or standby pump arrangements) is standard practice in the industry. Valid reason — not the answer.

Step 3 — Confirm the answer.

Choices B, C, and D are all legitimate reasons to use pumps in parallel. Choice A describes a benefit of series pumping, not parallel pumping. The question asks for the choice that is NOT a reason for parallel pumps.

✓ Answer: A — To increase pressure (this is what SERIES pumps accomplish, not parallel)

⭐ Key Takeaways
  • Parallel pumps add FLOW; series pumps add HEAD. This is the single most important distinction — commit it to memory. Confusing the two is the most common error on pump configuration questions.
  • Redundancy is a major practical reason for parallel installations. In real HVAC systems, parallel pumps are often sized so the system can operate at acceptable capacity even if one pump goes offline. This design intent shows up in PE exam conceptual questions.
  • "NOT" questions require you to evaluate every choice. Don't stop at the first answer that sounds plausible. The exam often includes multiple reasonable-sounding options, and the question is specifically testing whether you can identify the one that doesn't belong.

About This Problem Type

Parallel vs. series pump configuration questions are a recurring conceptual topic on the HVAC & Refrigeration PE exam. Unlike calculation-heavy problems, these test your understanding of fundamental hydraulic principles and your ability to reason through a "which is NOT" question format.

The core mental model: think of pumps in parallel as adding lanes to a highway — more cars (flow) can travel simultaneously, but the speed limit (pressure/head) doesn't change. Series pumps are like adding turbochargers in sequence — each stage boosts the pressure further, but the total volume moved per unit time doesn't increase.

On the PE exam, these conceptual questions often appear alongside affinity law problems. Make sure you can distinguish: Is this asking about changing pump speed/size? Or about how multiple pumps interact in a system?

"NOT" questions are a deliberate exam technique. They require you to identify the one incorrect statement among several true ones — testing deeper understanding rather than simple recall. Always process all choices before selecting.

Related Concepts
  • System Curve — the relationship between head and flow for the piping system; the operating point is where it intersects the pump curve
  • Best Efficiency Point (BEP) — the flow/head combination where the pump operates most efficiently; parallel staging keeps pumps near their BEP
  • Affinity Laws — govern how a single pump's performance changes with speed or impeller diameter; distinct from multi-pump configuration effects
  • Pump Curve Shape — the shape matters when combining pumps; steep curves and flat curves behave differently in parallel installations
  • NPSH Requirements — must be verified for each pump individually in a parallel arrangement; the suction conditions are shared

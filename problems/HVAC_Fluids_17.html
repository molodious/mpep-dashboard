Here you're getting double the volume flow rate if both pumps are running. If you turn one off, you get half. So the answer is A. You do not put pumps in parallel to increase the pressure. You do put them in parallel to increase the volume flow rate. Also, if you have variable speed drives, it's a good idea to put pumps in parallel to improve efficiency because if you have two pumps running at partial speed, that's more efficient than having one pump running at full speed. And this is true not only for an arrangement with two pumps, but suppose you have three or four or five pumps in parallel, you'd rather run all of them and slow them down to achieve the volume flow rate that you require. The more you have, the more efficient it becomes. Until you run it to some minimum, you may only be able to run the pumps so slow. So that sort of becomes the limit as to how far you can take that. And then for redundancy, parallel pumps helps that as well. A series pumps hurts because if any of these fail, they're all a single point of failure in delivering the flow to wherever it needs to go. But if you have pumps in parallel and one fails, the others will just speed up and do more. Or if they're running at full speed, maybe you don't have the FDs, but you still have more than enough pumping and you can afford to lose one if you have an n plus one design, let's say. So that is a potential reason you would put pumps in parallel for redundancy, efficiency, and to increase the flow rate, not to increase the pressure.

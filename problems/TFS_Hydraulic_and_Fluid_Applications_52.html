It's just going to go horizontally and then it's going to elbow up. Obviously that elbow adds a little bit of height but we can probably neglect that. And from there there's just a vertical discharge up to some height that we don't know and that's what we're interested in finding out. So it's labeled some of these points. We'll call the top of the tank state 1. We'll call the elbow state 2 or the discharge point state 2. And we'll just assume at that point is at 0 elevation at the very bottom. And then we want to know the final height that that water goes to state 3 which is sort of like almost like a projectile right? You're going upward from the discharge with some velocity.

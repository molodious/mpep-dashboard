Steam with a quality of 90% expands isentropically from 350 psia to 120 psia. What is the change in enthalpy?
A. 40 Btu/lb
B. 80 Btu/lb ✓
C. 90 Btu/lb
D. 120 Btu/lb
Correct Answer: B
⚡ Quick Overview
  1. Look up steam table values at P1 = 350 psia: gather hf, hfg, sf, and sfg.
  2. Calculate enthalpy at State 1: h1 = hf + χ·hfg = 409.8 + (0.9)(794.62) = 1125 Btu/lb
  3. Calculate entropy at State 1: s1 = sf + χ·sfg = 0.6059 + (0.9)(0.8914) = 1.408 Btu/(lbm·°F)
  4. Apply isentropic condition: s2 = s1 = 1.408 Btu/(lbm·°F)
  5. Look up steam table values at P2 = 120 psia; find quality at State 2: χ2 = (1.408 − 0.4919) / 1.0965 = 0.835
  6. Calculate enthalpy at State 2: h2 = 312.55 + (0.835)(878.2) = 1046 Btu/lb
  7. Find change in enthalpy: Δh = h1 − h2 = 1125 − 1046 = 79 Btu/lb → Answer B
📄 Full Solution

Look up the Properties of Saturated Water and Steam in the Reference Handbook organized by pressure and find 350 psia. Call this State 1. From the table, hf = 409.8 Btu/lb and hfg = 794.62 Btu/lb.

Calculate the enthalpy at State 1:

h1 = hf + χ·hfg = 409.8 + (0.9)(794.62) = 1125 Btu/lb

Also gather entropy values from the same line in the table:
sf = 0.6059 Btu/(lbm·°F) and sfg = 0.8914 Btu/(lbm·°F)

Calculate the entropy at State 1:

s1 = sf + χ·sfg = 0.6059 + (0.9)(0.8914) = 1.408 Btu/(lbm·°F)

Since the expansion from State 1 to State 2 is isentropic:

s2 = s1 = 1.408 Btu/(lbm·°F)

Gather entropy values from the table for State 2 at P2 = 120 psia:
sf = 0.4919 Btu/(lbm·°F) and sfg = 1.0965 Btu/(lbm·°F)

Find the quality at State 2:

χ = (s2 − sf) / sfg = (1.408 − 0.4919) / 1.0965 = 0.835

Gather enthalpy values for State 2: hf = 312.55 Btu/lb and hfg = 878.2 Btu/lb

Calculate the enthalpy at State 2:

h2 = hf + χ·hfg = 312.55 + (0.835)(878.2) = 1046 Btu/lb

Finally, calculate the change in enthalpy:

Δh = h1 − h2 = 1125 − 1046 = 79 Btu/lb

Answer: B

OH 115 March 2, 2026
Why can't we use Δh = Cp·ΔT to find enthalpy change between superheated steam and subcooled water?
Cp·ΔT only works within a single phase (sensible heating). When crossing phase boundaries — like melting or boiling — you must use enthalpy values directly because latent heat doesn't change temperature.
When does m·Δh become more useful than Cp·ΔT?
m·Δh is more powerful because it works across phase boundaries, while Cp·ΔT is limited to single-phase regions. The Cp value also differs for each phase (ice, water, steam), so using enthalpy directly avoids that complication.

Why isn't atmospheric pressure included in the stagnation pressure calculation for the submarine problem?
Atmosphere pushes down equally on both sides of the system, so it cancels out. Stagnation pressure is based on gauge static pressure only — you only add static pressure (gauge) and velocity pressure.
What's the best way to remember how stagnation pressure works?
Dan recommends understanding the pitot tube concept — stagnation pressure equals static pressure plus velocity pressure — rather than just memorizing the rule. The physical picture is more reliable under exam conditions.

How do you convert units from ft·lbf/(lbm·°R) to Btu/lb in a compressor work problem?
After computing R·ΔT/(1−k), the °R units cancel and you're left with ft·lbf/lbm. Convert to Btu using 1 Btu = 778 ft·lbf, which is found in the Reference Handbook measurement relationships.
Does ΔT in °R work the same as °F for unit conversions in these formulas?
Yes — for temperature differences (ΔT), Rankine and Fahrenheit are the same size, so the numerical value is identical. The distinction only matters for absolute temperature levels, not differences.

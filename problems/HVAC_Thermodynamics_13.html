HVAC & Refrigeration · Thermodynamics · Problem 13
Problem Statement
Steam with a quality of 90% expands isentropically from 350 psia to 120 psia. What is the change in enthalpy?
Answer Choices
A.40 Btu/lb
B.80 Btu/lb ✓
C.90 Btu/lb
D.120 Btu/lb
Solution
Answer: B
Quick Overview: 1. Identify State 1 as fully defined: P₁ = 350 psia, χ₁ = 0.90. Look up hf, hfg, sf, sfg from steam tables at 350 psia. 2. Calculate h₁ and s₁ using the two-phase mixture formulas: h = hf + χ·hfg and s = sf + χ·sfg. 3. Apply the isentropic condition: s₂ = s₁. This fully defines State 2 (along with P₂ = 120 psia). 4. Look up hf, hfg, sf, sfg from steam tables at 120 psia. Verify State 2 is still in the two-phase (saturated) region by checking sf < s₂ < sf + sfg. 5. Calculate quality at State 2: χ₂ = (s₂ − sf) / sfg. 6. Calculate h₂ = hf + χ₂·hfg, then find Δh = h₁ − h₂. Full Solution: Step 1 — Steam Table Values at State 1 (350 psia) Given: P₁ = 350 psia, χ₁ = 0.90 From saturated steam tables at 350 psia: hf = 409.8 Btu/lb hfg = 794.62 Btu/lb sf = 0.6059 Btu/(lbm·°R) sfg = 0.8914 Btu/(lbm·°R) Step 2 — Calculate h₁ and s₁ h₁ = hf + χ₁·hfg = 409.8 + (0.9)(794.62) = 409.8 + 715.16 = 1,125 Btu/lb s₁ = sf + χ₁·sfg = 0.6059 + (0.9)(0.8914) = 0.6059 + 0.8023 = 1.408 Btu/(lbm·°R) Step 3 — Apply Isentropic Condition Since the expansion is isentropic (reversible and adiabatic): s₂ = s₁ = 1.408 Btu/(lbm·°R) This, combined with P₂ = 120 psia, fully defines State 2. Step 4 — Steam Table Values at State 2 (120 psia) From saturated steam tables at 120 psia: hf = 312.55 Btu/lb hfg = 878.2 Btu/lb sf = 0.4919 Btu/(lbm·°R) sfg = 1.0965 Btu/(lbm·°R) sg = sf + sfg = 0.4919 + 1.0965 = 1.5884 Btu/(lbm·°R) Check: Is s₂ still in the saturated region? sf = 0.4919 < s₂ = 1.408 < sg = 1.5884 ✓ State 2 is confirmed to be in the two-phase (wet steam) region. Step 5 — Quality at State 2 χ₂ = (s₂ − sf) / sfg = (1.408 − 0.4919) / 1.0965 = 0.9161 / 1.0965 = 0.835 Step 6 — Calculate h₂ and Δh h₂ = hf + χ₂·hfg = 312.55 + (0.835)(878.2) = 312.55 + 733.3 = 1,046 Btu/lb Δh = h₁ − h₂ = 1,125 − 1,046 = 79 Btu/lb ≈ 80 Btu/lb → Answer B Why Not the Other Answers? • Answer A (40 Btu/lb): Approximately half the correct value — likely from using the wrong sfg or misreading quality. • Answer C (90 Btu/lb): Close, but results from rounding errors in intermediate steam table values. • Answer D (120 Btu/lb): Likely from skipping the quality recalculation at State 2 and assuming χ₂ = χ₁ = 0.9 throughout.
📹 Video Solution Synthesis:

• Dan's key insight: start with a "roadmap." State 1 is fully defined (pressure + quality known), so calculate everything you need there first — specifically h₁ and s₁ using the two-phase mixture formulas.

• The isentropic condition is the bridge between states: s₂ = s₁. This single fact, combined with P₂, fully defines State 2 — without it, you'd have only one property at State 2 and couldn't proceed.

• Quality changes between states. χ₁ = 0.90 but χ₂ = 0.835. A common mistake is using χ₁ at State 2. You must recalculate quality using entropy at State 2.

• Always verify you're still in the two-phase region at State 2: compare s₂ to sf and sg at the new pressure. Here sg at 120 psia ≈ 1.59 > 1.408, so we remain in the saturated region — the steam tables (not superheated tables) are correct to use.

• The core skill tested is steam table fluency: finding the right row (by pressure), reading the right columns (hf, hfg, sf, sfg), and plugging into the correct quality-based mixture formulas.
⭐ Key Takeaways:
• Two-phase mixture formulas: h = hf + χ·hfg and s = sf + χ·sfg. These are your primary tools whenever steam quality is given or needed. Memorize them — they appear constantly in thermodynamics problems.

• Isentropic = s is constant. For any isentropic process, s₂ = s₁. This is the bridge between states when pressure changes. Combine it with the known pressure at State 2 to fully define State 2.

• Quality changes between states. Never carry χ₁ forward to State 2 without recalculating. After finding s₂, solve χ₂ = (s₂ − sf)/sfg at the new pressure before computing h₂.

• Always verify phase at State 2. Check that sf < s₂ < sg at P₂. If s₂ > sg, the steam is superheated and you'd need the superheated steam tables — a completely different lookup path.

Q: When does it make sense to use hfg from the steam tables vs. using mCpΔT for heat calculations involving steam?
A: Use hfg (the latent heat of vaporization) whenever a phase change is involved — mCpΔT only works within a single phase (sensible heating or cooling) and completely misses the energy of phase transition. Across phase boundaries you must use enthalpy: hfg in the steam tables IS the latent heat of vaporization, and a single CpΔT calculation cannot span from subcooled water to steam because Cp is different in each phase. Q: How do I convert units like ft·lbf/(lbm·°R) to Btu/lb when working through thermodynamics problems?
A: Use the conversion factor 1 Btu = 778 ft·lbf, which is in the reference handbook's measurement relationships section. To convert ft·lbf/lbm to Btu/lb, divide by 778 — this comes up constantly in thermo problems involving the gas constant R and any work or energy calculation done in English units. (OH 32)

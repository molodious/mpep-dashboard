HVAC & Refrigeration · Thermodynamics · Problem 21
Problem Statement
An R-410A refrigeration cycle operates between 25.5 psia and 135 psia with 20°F of superheat and 20°F of subcooling. The compressor efficiency is 88%. What is the coefficient of performance?
Answer Choices
A.4.3 ✓
B.4.8
C.5.1
D.5.3
Solution
Answer: A
1. Look up R-410A saturation temperatures at 25.5 psia and 135 psia. 2. State 1: superheated vapor, T = T_sat(25.5 psia) + 20°F. Find h₁, s₁. 3. State 2s: isentropic compression to 135 psia (s₂s = s₁). Find h₂s. 4. State 2: actual compression with 88% efficiency: h₂ = h₁ + (h₂s - h₁)/0.88. 5. State 3: subcooled liquid, T = T_sat(135 psia) - 20°F. Find h₃. 6. COP = (h₁ - h₃) / (h₂ - h₁) ≈ 4.3 → Answer A.
This is a real-world vapor-compression cycle with both superheat, subcooling, and compressor inefficiency — a comprehensive PE exam problem. From R-410A property tables at 25.5 psia (evaporator) and 135 psia (condenser): State 1 (compressor inlet): T₁ = T_sat(25.5 psia) + 20°F superheat. Look up h₁ and s₁ from superheated R-410A tables. State 2s (ideal compressor outlet): isentropic compression to 135 psia. Find h₂s using s₂s = s₁ in the 135 psia superheated table. State 2 (actual compressor outlet): Account for 88% compressor efficiency. η_comp = (h₂s - h₁) / (h₂ - h₁) → h₂ = h₁ + (h₂s - h₁) / 0.88 State 3 (condenser exit): Subcooled liquid at T₃ = T_sat(135 psia) - 20°F. Approximate h₃ ≈ h_f at T₃. State 4 (after expansion): h₄ = h₃ (isenthalpic expansion). COP = q_L / w = (h₁ - h₄) / (h₂ - h₁) ≈ 4.3 → Answer A. The compressor inefficiency increases the work required (h₂ > h₂s), which reduces the COP compared to an ideal cycle. The subcooling increases the refrigerating effect slightly (lower h₃/h₄).

⭐ Key Takeaways:
• Compressor isentropic efficiency: η = (h₂s - h₁) / (h₂_actual - h₁). Rearrange to find actual exit enthalpy.

• Subcooling below condenser saturation temperature increases the refrigerating effect and improves COP slightly.

• Superheat at the compressor inlet ensures no liquid enters the compressor (protects equipment).

• Real vapor-compression cycles always have lower COP than ideal — compressor losses are the main contributor.

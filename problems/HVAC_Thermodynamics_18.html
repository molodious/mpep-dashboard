HVAC & Refrigeration · Thermodynamics · Problem 18
A refrigeration cycle using R-123 operates between 25 psia and 75 psia. Refrigerant enters the compressor with 20°F of superheat and leaves the condenser as a saturated liquid. What is the coefficient of performance?
  • A. 5.8
  • B. 6.7
  • C. 7.3
  • D. 10.5
✓ Correct Answer: B — 6.7

Quick Overview

  • Real refrigeration cycle — use enthalpy-based COP formula, not the Carnot temperature formula.
  • Four state points around the cycle: superheated vapor → compressed vapor → saturated liquid → two-phase mixture.
  • State 1 requires adding 20°F of superheat beyond Tsat at 25 psia to read h1 from the P-h diagram.
  • State 2 (h2) is the hardest read — a purely graphical isentropic compression. Small errors in h2 swing COP dramatically because the denominator is only ~8 Btu/lb.
  • COPR = (h1 − h4) / (h2 − h1) = 55.2 / 8.2 = 6.7.

Full Solution
State 1 — Compressor Inlet

Low-pressure side: P1 = 25 psia. From the R-123 saturation table, Tsat = 110°F. With 20°F of superheat, T1 = 130°F.

h₁ = 108.8 Btu/lb
State 2 — Compressor Outlet (Isentropic)

Follow constant-entropy lines from State 1 up to P2 = 75 psia on the P-h diagram.

h₂ ≈ 117 Btu/lb
State 3 — Condenser Outlet

Saturated liquid at P3 = 75 psia (quality χ = 0).

h₃ = 53.6 Btu/lb
State 4 — Evaporator Inlet

Isenthalpic expansion (throttling valve) to P4 = 25 psia.

h₄ = h₃ = 53.6 Btu/lb
COP Calculation
COP_R = Q_L / W = (h₁ − h₄) / (h₂ − h₁) = (108.8 − 53.6) / (117 − 108.8) = 55.2 / 8.2 = 6.73 ≈ 6.7 → Answer B

Key Takeaways
  • Use enthalpy-based COP for real cycles. COPR = (h1 − h4) / (h2 − h1). The Carnot temperature formula only applies to ideal cycles.
  • Small denominator = high sensitivity. When h2 − h1 is only ~8 Btu/lb, a 1 Btu/lb reading error changes COP by ~10%. Use the line tool and be precise.
  • Superheat shifts State 1. Don't forget to add the superheat to Tsat before reading h1 — it's 130°F, not 110°F.
  • Expansion is isenthalpic, compression is isentropic. h4 = h3 (throttling), and State 2 follows constant entropy from State 1.

Office Hours Q&A
Q: Why can't we just use temperatures to calculate COP, like in previous Carnot problems?
Previous problems were ideal Carnot cycles where COP = TL / (TH − TL). For real refrigeration cycles, you must use enthalpies from the P-h diagram: COPrefrigerator = (h1 − h4) / (h2 − h1). The Carnot formula only gives the theoretical maximum COP.

Q: What's the fundamental definition of COP, and how does it differ between heat pumps and refrigerators?
COP is "desired output over required input." For a refrigerator, the desired output is QL (cooling effect); for a heat pump, it's QH (heating effect). The required input is always compressor work W. The formulas: COPR = QL/W = (h1 − h4) / (h2 − h1); COPHP = QH/W = (h2 − h3) / (h2 − h1).

Q: I get h₂ = 118 instead of 117 from the R-123 chart, giving COP = 5.99 instead of 6.7. What should I do when being off by 1 Btu/lb changes the answer?
The denominator (h2 − h1) is small (~8–10 Btu/lb), so being off by 1 changes it by ~10%, swinging the whole COP significantly. Use the line tool on the exam, be as precise as possible on h2, but don't let it rattle you. Focus precision on the denominator.

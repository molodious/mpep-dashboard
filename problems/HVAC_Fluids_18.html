Problem
HVAC & Refrigeration · Fluids · Problem 18
Crude oil is pumped by a centrifugal pump with a performance curve given by the equation Hp = 50 − Q²/25 − Q, where head is in units of feet and flow rate is in units of cubic feet per second. The system curve is described by the equation Hs = Q²/25 + Q/10, with the same units. Neglecting efficiency and losses, what is the power required by the pump?
Answer Choices
A 16 hp
B 30 hp  ✓
C 35 hp
D 53 hp
Answer: B — 30 hp
Quick Overview
  1. Identify the operating point method: The pump operates where its performance curve intersects the system curve — set Hp = Hs.
  2. Set curves equal and rearrange: Combine the two equations into a single quadratic equation in Q.
  3. Solve the quadratic for Q: Use the quadratic formula; take the positive root. This gives the operating flow rate Q ≈ 19.05 cfs.
  4. Find the operating head H: Substitute Q back into either curve equation to get H ≈ 16.43 ft.
  5. Look up fluid property: For crude oil, use specific gravity SG ≈ 0.85, so γ = 0.85 × 62.4 = 53.04 lb/ft³.
  6. Compute power: Use P = γQH/550 (converts ft·lb/s to horsepower). Result ≈ 30 hp → Answer B.
📝 "Neglecting efficiency" means η = 1: When the problem says to neglect efficiency and losses, you treat the pump as 100% efficient. This means water (or fluid) power equals shaft power — no additional division by η is needed. The power formula P = γQH/550 gives shaft power directly in horsepower.

Full Solution

A centrifugal pump's performance curve (Hp) describes how much head the pump can deliver as a function of flow rate. The system curve (Hs) describes how much head the system demands at any given flow rate. The pump will naturally operate at the intersection of these two curves — the operating point.

Step 1 — Set Hp = Hs:

H_p = H_s 50 − Q²/25 − Q = Q²/25 + Q/10

Step 2 — Rearrange to standard quadratic form:

Move all terms to one side:

50 = Q²/25 + Q/10 + Q²/25 + Q 50 = 2Q²/25 + Q(1 + 1/10) 50 = 2Q²/25 + 11Q/10

Multiply every term by 50 to clear fractions:

2500 = 4Q² + 55Q 4Q² + 55Q − 2500 = 0

Step 3 — Apply the quadratic formula:

Q = [−b ± √(b² − 4ac)] / (2a) a = 4, b = 55, c = −2500 discriminant = 55² + 4(4)(2500) = 3,025 + 40,000 = 43,025 √43,025 ≈ 207.42 Q = (−55 + 207.42) / (2 × 4) = 152.42 / 8 ≈ 19.05 cfs

(The negative root gives a physically meaningless negative flow rate — discard it.)

Step 4 — Find the operating head H:

Substitute Q = 19.05 cfs into the system curve:

H = Q²/25 + Q/10 H = (19.05)²/25 + 19.05/10 H = 362.9/25 + 1.905 H = 14.52 + 1.91 ≈ 16.43 ft

(Verify with pump curve: H_p = 50 − (19.05)²/25 − 19.05 = 50 − 14.52 − 19.05 = 16.43 ft ✓)

Step 5 — Fluid specific weight for crude oil:

Crude oil has a specific gravity of approximately SG = 0.85 (a standard PE exam value; crude typically ranges 0.80–0.97).

γ = SG × γ_water = 0.85 × 62.4 lb/ft³ = 53.04 lb/ft³

Step 6 — Compute required power:

With η = 1 (neglecting efficiency), fluid power equals shaft power:

P = γ × Q × H / 550 P = 53.04 lb/ft³ × 19.05 ft³/s × 16.43 ft / 550 P = 53.04 × 312.8 / 550 P = 16,591 ft·lb/s / 550 P ≈ 30.2 hp

Answer: B — 30 hp.

The factor of 550 converts from ft·lb/s to horsepower (1 hp = 550 ft·lb/s). Since efficiency is neglected (η = 1), there is no denominator efficiency term — every unit of power delivered to the fluid equals one unit of shaft power drawn by the pump.

📋 Section 1 — Quick Overview
  1. Identify the operating point condition: The pump delivers exactly the head the system demands — find Q where Hp = Hs.
  2. Set curves equal and simplify: Substitute both equations and rearrange into a standard quadratic: 4Q² + 55Q − 2500 = 0.
  3. Solve the quadratic: Apply the quadratic formula to get Q ≈ 19.05 cfs (take positive root only).
  4. Calculate operating head: Plug Q back into either curve → H ≈ 16.43 ft.
  5. Determine fluid specific weight: Crude oil SG ≈ 0.85 → γ = 53.04 lb/ft³.
  6. Compute power: P = γQH / 550 ≈ 30 hp (Answer B).
📐 Section 2 — Full Solution

This is a classic pump operating point problem. Two curves are given — a pump performance curve that shows decreasing head with increasing flow (typical centrifugal pump behavior), and a system curve that shows increasing head demand with increasing flow (friction and elevation losses).

The operating point is where the pump can deliver exactly the head the system requires. Mathematically, we set Hp = Hs and solve for the flow rate Q. Both curves contain Q² terms, so when combined, they form a quadratic equation.

After solving 4Q² + 55Q − 2500 = 0, the positive root gives Q ≈ 19.05 cfs. Plugging this back into either curve confirms the operating head H ≈ 16.43 ft. Both curves agree — this validates the solution.

The power calculation uses the standard hydraulic power formula P = γQH, converted to horsepower by dividing by 550 (since 1 hp = 550 ft·lb/s). Since the problem says "neglecting efficiency," we use η = 1, meaning fluid power equals shaft power directly.

The crucial detail many students miss: this is crude oil, not water. The specific weight γ must be adjusted using the specific gravity of crude oil (SG ≈ 0.85), giving γ = 53.04 lb/ft³ instead of 62.4 lb/ft³.

📺 Section 3 — Video Synthesis
Key Concepts for Pump Curve Problems
  • The operating point of any pump-system combination is always at the intersection of the pump performance curve and the system curve — this is a fundamental concept you must know cold for the PE exam.
  • Setting Hp = Hs and rearranging almost always yields a quadratic equation in Q when both curves contain Q² terms — be prepared to use the quadratic formula quickly.
  • The power formula P = γQH / 550 (in US customary units) applies when η = 1. If efficiency η is given, shaft power becomes P = γQH / (550 × η).
  • For fluid properties: always check whether the problem involves water (γ = 62.4 lb/ft³) or another fluid. For crude oil, use SG ≈ 0.85 unless the problem specifies otherwise.
  • In SI units the power equation is P = γQH (in watts, with γ in N/m³, Q in m³/s, H in m). Know both unit systems for the PE exam.
  • A typical centrifugal pump curve shows head decreasing as flow increases. System curves show head increasing with flow (friction ∝ Q²). Their intersection defines the unique stable operating point.
⚡ Section 4 — Key Takeaways
  • The operating point is always at the intersection of the pump curve and system curve — set them equal, solve for flow rate, then find head.
  • On the PE exam, pump problems involving non-water fluids almost always require you to adjust the specific weight — never blindly use 62.4 lb/ft³ without checking the fluid.
  • "Neglecting efficiency" is exam shorthand for η = 1, which simplifies the power calculation — no extra denominator, no guessing at pump efficiency.
  • When the system curve and pump curve both contain Q² terms, expect to clear fractions and use the quadratic formula — practice this algebra quickly.

And then once they recombined, the total volume flow rate is going to be 40 GPM once again. And we don't know the relative volume flow rates. Right. We don't know how it splits up between A and B. And that's kind of what we're after in this problem. The question being asked is what's the flow rate through Branch A. So we'll call that QA. That's what we're looking for. And as is often the case with parallel piping questions, we want to start off by equating the head loss through each branch. And that would be true even if there were multiple branches. So in this case, there's just two. But if there were three or four or more, then we could say that the head loss through each branch is the same as every other branch. So we'll use the Darcy equation to do that. We'll start by saying the head loss through Branch A is equal to the head loss through Branch B. And then we'll expand on that by saying F, L, V over 2 D, G is the same for A as it is for B. And now we can make some convenient simplifying assumptions. We can cancel out the constants that are on both sides. So that includes 2 and G. We also know that the diameter is 2 inch steel pipe for Branch A and Branch B. So since that's the same, the diameter can be canceled out. And then the friction factor depends on a couple of things. It depends on the relative roughness. But they're both steel pipe. So that's going to be the same. And then it also depends on the Reynolds number, which is equal to velocity times diameter over kinematic viscosity. Diameter and kinematic viscosity aren't changing. So the only thing that's going to change is the velocity is going to be slightly different through each of the two branches. But is a slight difference in velocity going to change the Reynolds number enough that when you go and use the moody diagram to look up the friction factor that the friction factor is going to change significantly, probably not. So we can make the assumption here that since these flow rates are going to be in the same ballpark, their velocities are going to have the same at least order of magnitude. Therefore, the friction factors will be approximately equal. So that's a key assumption that's going to help us simplify here. So then what does that mean?

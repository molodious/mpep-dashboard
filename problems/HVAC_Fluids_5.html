Full Solution: Fluids number 5, a pump designed to operate at a 100gpm and 50 feet of head is determined to be oversized. The pump's impeller diameter is decreased by 20%. What is the resulting pressure developed by the pump after this change? So this idea of changing one of the parameters of a pump and trying to figure out how the pressure changes or how the power changes is best handled using the pump affinity loss. So let's jump right into searching the reference handbook for pump affinity. So that brings us to this table and this is a great place to look. We can see that when you change the speed which they're using variable n to show this, that has a certain impact on the flow, right? It's linear with flow and the same for the diameter if you change the diameter that is linear with the flow rate. However, the impact of changing the speed on the head pressure is not linear, it's actually squared. And the impact on the power is that it's cubed. So in this case we're going to be changing the diameter so this is the relationship we're going to look at. But before we go applying any equations, let's just get a handle on what we know here. We have some initial conditions. We have an initial flow rate Q1 which is known 100gpm. We also have the initial head pressure. We'll call that delta H1 and these are both known. There's also some initial diameter. We don't know what it is. But we know that the new diameter is only 80% of the original diameter because it's been reduced by 20%. So we could say d2 over d1 equals 0.8. Then presumably by making that change, there would be some change to the flow rate and there would be some change to the delta H. So there'd be a flow rate 2 and a delta H2. And we're not particularly interested in what the new flow rate is going to be. We are interested in what the new pressure is going to be. That's what we've been asked, what's the resulting pressure developed by the pump after the change. So now let's go back to our formula. H2, the new head pressure is the old head pressure times the ratio of the new diameter to the old diameter squared. And that's exactly what we're going to apply. So H2 over H1 is d2 over d1 squared. And if we want to get H2 by itself, the chart already showed this. But I like to show it this way because that's how I remember it. You can get H2 by itself. It's H1 and I'm not writing the delta, but it is a delta P if you like to think of it that way. d2 over d1 quantity squared. And the original pressure was 50 feet. And the ratio of the new diameter to the old diameter, it's smaller. It's only 80% because it was deemed to be oversized. So 0.8 squared. And that works out to 32 feet, which is answer choice B. Had it been linear, it would have went down from 50 to 40, but it's not. It's related to the square.

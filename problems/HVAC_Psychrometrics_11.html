What is the dew point of mixture? So the way we're going to go at this, the dew point is not a linear number, right? It arrives along the saturation curve of the psychometric chart. So we can't find the dew point of each stream and then try to mix them together because of that non-linearity. What we can do is we can find the humidity ratio of each stream, the stream coming over the coil and the stream which is bypassed and then we can do a mixing calculation on the humidity ratio which is linear. And then we can use the fact that the humidity ratio basically implies the dew point temperature and let's illustrate that by drawing a little psychometric chart here. So along the vertical axis we have the humidity ratio and along the saturation curve we have the dew point. So at some point where you know the humidity ratio, you can go horizontally to the right to find that you can go horizontally to the left and it should just be one continuous line and you can find the dew point temperature the same way. So we're going to focus on the humidity ratio and the solution and once we find it we'll just move horizontally across the psychometric chart to find the corresponding dew point.
